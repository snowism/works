Snkrs ZAP!

Project Overview

The SNKRS ZAP is a C2C application to buy and sell for all lovers of sneakers.
The app is created by MERN stack will allow users to define the sneaker they are looking for via brand name, price, main colour, shoe size, new/used shoe and style. The sneaker will then be able to be added to a shopping cart or removed from a shopping cart, if more than one of the same sneakers is wanting to be purchased they should be able to adjust the cart accordingly.

I was responsible for all the User Research, Personas and Development.
